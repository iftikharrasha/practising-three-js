Formschön - Shoe shop

Type: XD to HTML

Formschön was another foreign project i did for a german fashion designer. I will remember this project for so long. The client provided me an XD file and the job was to execute her design sense with my coding. Not only i had to convert her complex designing into code but also i had to make it fast and fully responsive to all the devices. For this i used React JS and implemented lazyloading, routing, smooth scrolling, interactive animations and so on.
